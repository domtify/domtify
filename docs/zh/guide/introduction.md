一个现代化的真正支持单个方法树摇(Tree-shaking)优化的 dom 操作库,与[jQuery](https://github.com/jquery/jquery)
有着类似的 API。

## 从 jQuery 迁移

和 jQuery 完全保持一致不是目标，可能会存在细微的差别(我把这部分称为优化),如果您要从 `jQuery` 迁移，请务必阅读我们的[迁移指南](./migration.md)。

## 包名的由来

包名 domtify 的灵感来源于基于 vue 的 ui 组件库[vuetify](https://github.com/vuetifyjs/vuetify)

- vuetify = vue+tify
- domtify = dom+tify

分享包名的由来只是为了能让您的印象更加深刻。

## 浏览器支持

domtify 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。
