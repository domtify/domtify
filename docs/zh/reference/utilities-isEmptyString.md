# d.isEmptyString()

## d.isEmptyString( value )

- **返回:** [Boolean](/reference/Types#boolean)

### d.isEmptyString( value ) <small style="font-size: 12px; color: var(--vp-c-green);">from @1.0</small>

- **value**
  - **类型:** [Anything](/reference/Types#anything)

