# d.isBoolean()

## d.isBoolean( value )

- **返回:** [Boolean](/reference/Types#boolean)

判断值是布尔值。

### d.isBoolean( value ) <small style="font-size: 12px; color: var(--vp-c-green);">from @1.0</small>

- **value**
  - **类型:** [Anything](/reference/Types#anything)
  - **描述:** 任意类型的参数

```js
d.isBoolean(true) // => true
d.isBoolean(false) // => true
d.isBoolean("") // => false
d.isBoolean(null) // => false
```
