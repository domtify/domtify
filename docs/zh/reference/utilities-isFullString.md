# d.isFullString()

## d.isFullString( value )

- **返回:** [Boolean](/reference/Types#boolean)

判断值是非空字符串。

### d.isFullString( value ) <small style="font-size: 12px; color: var(--vp-c-green);">from @1.0</small>

- **value**
  - **类型:** [Anything](/reference/Types#anything)
  - **描述:** 任意类型的参数

```js
d.isFullString("  ") // => true
d.isFullString("") // => false
```
