# d.isEmptyArray()

## d.isEmptyArray( value )

- **返回:** [Boolean](/reference/Types#boolean)

判断值是空数组。

### d.isEmptyArray( value ) <small style="font-size: 12px; color: var(--vp-c-green);">from @1.0</small>

- **value**
  - **类型:** [Anything](/reference/Types#anything)
  - **描述:** 任意类型的参数

示例：

```js
d.isEmptyArray([]) // => true
d.isEmptyArray([1, 2, 3]) // => false
```
