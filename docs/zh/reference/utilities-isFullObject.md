# d.isFullObject()

## d.isFullObject( value )

- **返回:** [Boolean](/reference/Types#boolean)

判断值是一个非空对象。

### d.isFullObject( value ) <small style="font-size: 12px; color: var(--vp-c-green);">from @1.0</small>

- **value**
  - **类型:** [Anything](/reference/Types#anything)
  - **描述:** 任意类型的参数

```js
d.isFullObject({ foo: "bar" }) // => true
d.isFullObject({}) // => false
```
