# d.isPositiveNumber()

## d.isPositiveNumber( value )

- **返回:** [Boolean](/reference/Types#boolean)

判断值是正数。

### d.isPositiveNumber( value ) <small style="font-size: 12px; color: var(--vp-c-green);">from @1.0</small>

- **value**
  - **类型:** [Anything](/reference/Types#anything)
  - **描述:** 任意类型的参数

```js
d.isPositiveNumber(1) // => true
d.isPositiveNumber(3.14) // => true

d.isPositiveNumber(0) // => false
d.isPositiveNumber(-1) // => false
```
