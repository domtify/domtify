# d.isUndefined()

## d.isUndefined( value )

### d.isUndefined( value ) <small style="font-size: 12px; color: var(--vp-c-green);">from @1.0</small>

- **value**
  - **类型:** [Anything](/reference/Types#anything)

```js

d.isUndefined(null) // => false
```
