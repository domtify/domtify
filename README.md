一个现代化的真正支持单个方法树摇(Tree-shaking)优化的dom操作库,与[jQuery](https://github.com/jquery/jquery)
有着类似的API。

## 文档

要查看文档，请访问 [domtify.com](https://www.domtify.com/)。
